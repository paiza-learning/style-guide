## 関数の引数を書き換える (out 的なパターン) は避ける

このパターンは基本的に避けてください。

基本的には関数の戻り値を利用し、破壊的なものならばクラスのメソッドとして実装するのが丸いと思われます。

```c++
// ちょっと上手いこと書くので待ってね
```
